# 🌐 Réseau Fédéré : Collaboration Université–Entreprise
> Optimiser le Partage et la Sécurité

## 📋 Description
Projet agile visant à créer un réseau fédéré sécurisé 
permettant aux universités et entreprises de partager 
leurs ressources tout en préservant leur autonomie.

## 🎯 Problématique
Les universités et entreprises fonctionnent de manière 
isolée. Ce projet propose un système VPN fédéré avec 
authentification SSO et chiffrement des données.

## 🛠️ Technologies utilisées
- VPN (réseau sécurisé)
- SSO (authentification unique)
- Protocoles : OSPF, BGP, HTTPS
- Outil agile : Trello
- Gestion de code : GitHub
